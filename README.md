# Pomodori es Tomatoes!
Basic pomodoro app based on Roberto Fernandes' [YouTube video](https://www.youtube.com/watch?v=pATGCf191to).

<b><h3 align="left">Properties:</h3></b>

- System default / Dark mode / Light mode switch.
